# SenseEd AI Assistant

Multimodal accessibility and educational assistant designed for deaf and mute students using:
- Sign Language Recognition
- Speech Processing
- Computer Vision
- AI-based Interaction

---

## Features

- Sign language detection
- Speech-to-text
- YOLO object detection
- Educational assistance
- GUI interaction

# System Architecture

```text
Camera Input ──► Sign Language Module ──► Text Output
Microphone ───► Speech Module ─────────► GUI Interface
Vision Module ─► Object Detection ────► User Interaction
```
