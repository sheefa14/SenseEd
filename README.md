# SenseEd

AI-powered multimodal accessibility assistant for deaf and mute students.

## Features

- Sign language detection
- Speech-to-text
- YOLO object detection
- Educational assistance
- GUI interaction

## Technologies

- Python
- OpenCV
- YOLOv8
- PyTorch
- Tkinter

## Run Project

```bash
python main.py
```
